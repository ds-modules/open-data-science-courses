# Data 8 (edX)

Information and resources for the **Data 8** edX course.

**Course Overview:** Data 8: Foundations of Data Science (edX version) offers an accessible introduction to data science concepts and computational thinking. The course covers Python programming basics, data manipulation and analysis, statistical inference, and data visualization. Students work with real datasets to develop skills in exploratory data analysis, hypothesis testing, and drawing conclusions from data. Designed for learners without extensive programming background, this course provides the fundamental tools and mindset needed for data-driven problem solving and further study in data science.

## How to Access

This course may be available on edX. For the latest availability, search for "Data 8" on the [edX platform](https://www.edx.org/search?q=Data+8).

## Source and Materials

This edX course is based on UC Berkeley's Data 8: Foundations of Data Science. The official Berkeley site is [data8.org](https://data8.org/), which hosts the publicly licensed textbook, assignments, and lecture materials that much of this content is adapted from or closely follows.
