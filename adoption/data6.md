# Data 6

Materials and adoption information for **Data 6**.

**Course Overview:** Data 6: Foundations of Data Science introduces students to the fundamental concepts and techniques of data science. The course covers Python programming for data manipulation, exploratory data analysis, data visualization, and basic statistical inference. Students learn to work with real-world datasets, apply computational thinking to solve problems, and communicate insights effectively. This course serves as an entry point for students interested in data science, providing the computational and statistical foundations needed for more advanced study.
